# PopX React Assignment

A React + Vite implementation of the PopX UI assignment.

## 🚀 Features

- Landing Page
- User Signup
- User Login
- Local Storage Authentication
- Dynamic Profile Page
- Logout Functionality
- Responsive Mobile UI

## 🛠️ Tech Stack

- React
- Vite
- React Router DOM
- CSS3

## 📦 Installation

```bash
npm install
```

## ▶️ Run Locally

```bash
npm run dev
```

## 📁 Folder Structure

```
src/
 ├── assets/
 ├── components/
 ├── pages/
 ├── styles/
 ├── App.jsx
 └── main.jsx
```
